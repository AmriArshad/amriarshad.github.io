235Flix

Built a basic web application as a COMPSCI 235 assignment that relies on a repository pattern and other architectural design patterns such as Dependency Inversion and Single Responsibility. The app utilises the Flask framework and uses Jinja templating and WTForms. Flask Blueprints have also been used to maintain some level of seperation between the apps functions. All testing has been done using pytest and includes unit and end-to-end integration testing.
The github repo can be found here.
